# Quality-Filter Study — setup detectors over daily candles

Generated: 2026-07-16T18:17:34.180Z | args: --days 180 --holdBars 5
Cost: 0.05%/side. Simulation identical to backtest_setups.ts (honest fills, stop-first tie-break).

Filters (all computed from bars <= signal bar only):
- **TREND** — close > EMA50 (longs) / close < EMA50 (shorts)
- **VOLUME** — signal-bar volume >= 1.5x prior-20-bar average
- **RSI** — reject RSI14 > 70 longs / RSI14 < 30 shorts
- **RS** — 60-bar return minus equal-weight universe mean (same date); > 0 longs, < 0 shorts
- **STOPATR** — |entry - stop| / ATR14 >= 0.8

`expectancyPct` = avg net return per **filled** trade (timeouts included, no-fills excluded).
`winRatePct` = wins/(wins+losses), ignores timeouts. `retention` = signals kept vs baseline.
Note: filters gate the baseline signal stream post-hoc; live cooldown sequencing after a rejected signal may differ slightly.
